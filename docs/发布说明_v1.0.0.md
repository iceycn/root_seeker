# RootSeeker v1.0.0 发布说明

**发布日期**：2025-03-05

RootSeeker 首个正式版本，面向公司内网的 AI 驱动错误分析与根因发现服务。

---

## 核心能力

- **错误日志接入**：Webhook 接收、阿里云 SLS 主动补全、批量聚类
- **多源检索**：Zoekt 词法检索 + Qdrant 向量检索，支持代码与日志关联
- **AI 分析**：对接 DeepSeek/豆包等 LLM，自动生成根因分析与建议
- **通知推送**：企业微信、钉钉 Webhook（支持 sign/keyword/IP 三种安全模式）
- **管理端**：RootSeeker Admin（若依），仓库管理、索引触发、配置管理

---

## 主要功能

| 模块 | 说明 |
|------|------|
| **日志接入** | `POST /ingest`、`POST /ingest/aliyun-sls`、`POST /ingest/batch-cluster` |
| **Git 仓库发现** | 统一接口获取 Gitee/GitHub/Codeup 等平台仓库列表，支持 HTTPS 拉取 |
| **索引管理** | 向量索引、Zoekt 索引、依赖图重建、全量同步/清除 |
| **事件化流程** | 索引、移除、重新同步等事件入队，支持异步执行 |
| **配置热重载** | 配置变更提示 |

---

## 部署方式

| 方式 | 适用场景 |
|------|----------|
| **macOS / Linux** | `bash scripts/install-without-docker.sh` + `bash scripts/start-all-one-click.sh` |
| **Windows** | `.\scripts\install-without-docker.ps1` + `scripts\start-all-one-click.bat` |
| **Docker** | `cd root_seeker_docker && docker compose up -d`（全栈：MySQL + Qdrant + Zoekt + RootSeeker + Admin） |
